# Inventory Management System (C#)

## Overview

This project is a console-based Inventory Management System written in C#. It was designed to add, update view and delete inventory items from and to a file.

## Desccription

It does all of the following tasks:

- Add products(Id, Name, Category, Note/Description, Quantity, and Price)
- View Products
- Update product
- Delete product
- Save inventory to a file
- Load inventory from a file
